API Documentation

1. Messages API

GET /api/messages

Description: Get all messages

Headers: Authorization: Bearer <token>

Responses:

200 OK
[ { "id": "message_id", "fullname": "John Doe", "email": "john.doe@example.com", "subject": "Subject", "message": "Message content", "ipAddress": "IP_ADDRESS", "createdAt": "timestamp" }, ... ]
401 Unauthorized
{ "error": "Unauthorized" }

GET /api/messages/:id

Description: Get a message by ID

Headers: Authorization: Bearer <token>

Responses:

200 OK
{ "id": "message_id", "fullname": "John Doe", "email": "john.doe@example.com", "subject": "Subject", "message": "Message content", "ipAddress": "IP_ADDRESS", "createdAt": "timestamp" }
404 Not Found
{ "error": "Xabar topilmadi" }
401 Unauthorized
{ "error": "Unauthorized" }

DELETE /api/messages/:id

Description: Delete a message by ID

Headers: Authorization: Bearer <token>

Responses:

200 OK
{ "message": "Message successfully deleted" }
404 Not Found
{ "error": "Xabar topilmadi" }
401 Unauthorized
{ "error": "Unauthorized" }
